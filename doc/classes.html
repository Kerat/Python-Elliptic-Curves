bezout(a, b)¶

Return (u, v, p) of the equation : a*u + b*v = p. p is the gcd of a and b. :param a: First parameter. :param b: Second parameter. :return: (u, v, p).

double(point)¶

Compute the double of a point (multiply it by two). :param point: The point to double (a 3 elements tuple). :return: The double of the point given in parameter (a 3 elements tuple).

inv_modulo(x, m)¶

Return the symmetric of x modulo m :param x: parameter wich we want to find the symmetric. :param m: modulo in wich the symmetric will be calculate. :return: Return the symmetric, raise an exception if the numbers are not primes between each other.

multiply(point, n)¶

Multiply by n a given point. :param point: The point to multiply (a 3 elements tuple). :param n: The factor of the multiplication (an integer) :return: The result point of the multiplication (a 3 elements tuple).

opposite(point)¶

sum(point1, point2)¶
